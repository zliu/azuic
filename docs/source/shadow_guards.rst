军团简介
--------
银河影卫是银河下属的一支军团，游戏内本军团和银河其他军团无任何从属关系。

影卫目前的日常活动是在虫洞空间中进行的，军团驻地也位于虫洞内。

军团现有2个虫洞空间

萌新游乐园：2-1-7， 这是一个二级虫洞空间，空间内有2个固定虫洞，一个通往一级虫洞空间，另一个通往高安地区。这个空间主要用于接收技能不高，经验不足的萌新，可以在洞内扫扫信号，挖挖坟，采采气，种种菜

老司机竞技场：2-3-7，这是一个二级虫洞空间，空间内有2个固定虫洞，一个通往三级虫洞空间，另一个通往高安地区。这个空间主要用于给老司机飙车，可以做你爱做的事情

虫洞战斗职业路线
^^^^^^^^^^^^^^^^

虫洞作业中的战斗分为两种，最主要的一种就是刷怪，也是获得收益的主要途径，刷怪技能是为了击毁各个信号点的各等级怪，并且需要能抵抗住虫洞怪的各种抗性打击，还包括对抗怪的网子，毁电，遥修等装备

另一种次要的战斗技能，是用于和其他玩家对抗的，通常虫洞中的生产作业等，需要尽量避免和闯入的陌生玩家发生冲突，发现其他玩家闯入一般是走为上策，然而有时候如果需要对闯入的玩家进行抓捕或者驱赶，就需要有和玩家战斗（PVP）的技能了

虫洞挖坟职业路线
^^^^^^^^^^^^^^^^

虫洞挖坟作业，所需要的基础技能就是扫描探针的使用技能，扫描探针用来对虫洞内的一切信号进行定位，包括气云，数据点，遗迹点，虫洞点

在掌握了扫描技能后，如果选择挖坟路线，就需要学习破解数据点和遗迹点的技能，进而可以在破解后打开宝箱

对应虫洞战斗路线，扫描技能在虫洞内的另一个不常用的领域就是抓捕或驱赶其他玩家时，需要用扫描技能对目标玩家进行精确定位，从而进一步实施打击

虫洞后勤职业路线
^^^^^^^^^^^^^^^^

这里的后勤，指的是军团的生产后勤，负责科研、制造、运输、销售、采购以及军团管理等事务，而不是作战时舰队中对战斗船只进行辅助的后勤舰船作业。

虫洞后勤作业，包括对虫洞内的矿石点，气云点进行矿石和气云的采集，还有在军团POS内进行图纸研究，舰船及装备的制造

其中研究作业分为对图纸进行材料效率研究，时间效率研究，将T1原图复制获得复制图纸，对T1复制图纸进行发明获得T2图纸，以及对T3舰船及装备的残骸（上述挖坟作业中获取到）进行逆向工程获得T3图纸

制造作业分为对T1，T2，T3舰船及装备图纸进行制造，以及某些舰船和装备制造所需原材料中的‘制造组件’进行制造。制造舰船，装备，组件所需的原材料中，有些是矿石提炼的矿物，有些则是由行星产物，冰矿或气云等物品经过复杂加工工序获得的，而这些加工作业，也是生产制造作业中涵盖的一部分作业内容

后勤职业对玩家的技能要求是非常高的，尤其是能给军团带来收益的T2和T3类高级制造，在大量技能满级之前，能做到的只是对初级图纸的研究和初级物品的制造，并不能给军团带来任何收益，因此后勤职业在人物技能成型之前会有很长一段时间是无法给玩家自己带来显著收益的，这点和上述战斗及挖坟职业相比，有较大差别，非后勤类的职业，能够很快具备作业技能，并且跟队活动，获取可观收益，但是大部分物资补给是源于军团后勤部门的产出的。后勤职业相对其他职业可能会明显枯燥。

种菜
^^^^
行星开发，俗称种菜，是个相对独立的玩法，上述任何职业路线都可以捎带学习种菜技能，在行星上布置好资源采集设施后可以连续多天无需关注。加之行星开发技能总共只有5个，倍增系数也不高，因此种菜这一项玩法算不上什么职业。不过行星开发所得到的产物，是虫洞其他作业尤其是制造业不可或缺的资源。
